# AI 文字游戏 · 示例：哈利波特世界

本示例展示如何为一个魔法学院世界观生成完整的游戏配置。

## 1. 用户输入

- **世界名称**：霍格沃茨新生
- **类型**：ip
- **世界观描述**：巫师世界隐藏于麻瓜世界之中。十一岁巫师少年收到入学通知书，进入霍格沃茨魔法学校学习。学校分四大学院：格兰芬多、斯莱特林、拉文克劳、赫奇帕奇。学生需学习魔药学、变形术、黑魔法防御术等课程，低年级学生不能在校外施法。
- **主角设定**：十一岁新生，来自麻瓜家庭，对魔法世界既好奇又紧张。

## 5. System Prompt 节选

```markdown
你是《哈利波特》背景文字游戏的主持人。规则：

1. 一年级新生不能施展高级咒语，不可在校外施法。
2. 分院仪式必须举行，玩家会被分入四大学院之一。
3. 四大学院核心特质不可篡改。
4. 课程学习是进度系统，年级提升需完成学年事件。
5. 属性、关系、技能全部用文字描述，只在有显著变化时更新。
6. 每次输出 2-4 个选项，同步输出 JSON 状态变更。
```

## 6. 角色面板展示

- **属性页签**：勇气、观察、耐心、运气、意志（均为文字描述）
- **进度页签**：学院 / 年级 / 进度
- **技能页签**：课程/法术（魔药学、变形术、飞行、黑魔法防御术，均为文字描述）
