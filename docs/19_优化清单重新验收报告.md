## 验收口径

原清单中的“已完成”全部作废重验。本报告只接受三类证据：可复现自动测试、完整数据链路检查、真实浏览器交互。仅存在函数名、注释或按钮占位不算完成。

统一验证命令：

```bash
npm run verify
```

该命令依次执行内联脚本语法、ES Module import/export、真实模块加载和 Node 单元/集成测试。
